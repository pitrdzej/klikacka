# Klikacka — local development & deploy

- Local dev (inside `klikacka`):

```bash
cd klikacka
npm install
npm run dev
```

- Build and preview:

```bash
cd klikacka
npm run build
npm run preview
```

- Deploy (script copies `klikacka/dist` files to repository root so static files are served from root):

```bash
npm run deploy
```

Notes:
- The `assets/`, `sounds/` and `bosses/` folders in the repository root are generated by the `deploy` script. If you rebuild and re-run `npm run deploy` they will be updated.
- `klikacka/vite.config.ts` uses a relative `base` so built files work both when served from root and when copied into subfolders.
